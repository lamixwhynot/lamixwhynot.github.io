# 🌸 Сайт для любимой — инструкция

## Структура файлов
```
loveweb/
├── index.html       ← главная (сердце + таймер)
├── moments.html     ← галерея фото
├── letter.html      ← письмо + причины
├── style.css        ← все стили
├── script.js        ← анимация + таймер
├── photos/          ← СЮДА клади свои фотографии
│   ├── photo1.jpg
│   ├── photo2.jpg
│   └── ...
└── README.md
```

## ✏️ Что нужно поменять

### 1. Имя девушки (index.html)
Найди строку:
```html
<h1 class="hero-name">Анечка</h1>
```
Замени `Анечка` на её имя.

---

### 2. Дата начала отношений (script.js)
Найди строку:
```js
const START_DATE = new Date(2024, 0, 15, 12, 0);
```
Формат: `new Date(год, месяц-1, день, часы, минуты)`

Примеры:
- 14 февраля 2023 → `new Date(2023, 1, 14, 0, 0)`
- 7 марта 2024    → `new Date(2024, 2, 7, 0, 0)`
- 25 декабря 2022 → `new Date(2022, 11, 25, 0, 0)`

> ⚠️ Месяц считается с нуля: январь=0, февраль=1, ..., декабрь=11

---

### 3. Фотографии (moments.html)
1. Скопируй фото в папку `photos/`
2. В `moments.html` найди каждый блок с фото и раскомментируй `<img>`:
```html
<!-- <img src="photos/photo1.jpg" alt="наш первый день"> -->
```
→ убери `<!--` и `-->`:
```html
<img src="photos/photo1.jpg" alt="наш первый день">
```
3. Удали или скрой блок `<div class="ph">...</div>` под ним.
4. Поменяй подписи в `<div class="photo-caption">`.

---

### 4. Личное письмо (letter.html)
Найди блок `<div class="paper-body">` и замени текст на своё.
Ищи комментарий `✏️ ЗАМЕНИ`.

### 5. Причины (letter.html)
Замени текст в `<span class="r-text">` на свои причины.
Можно добавить/удалить пункты — просто копируй/удаляй блоки `<li class="r-item">`.

### 6. Подпись (letter.html)
```html
<div class="paper-sign">твой Артём ♡</div>
```
Замени `Артём` на своё имя.

## 🚀 Деплой на GitHub Pages (бесплатно)

### Шаг 1 — создай репозиторий
1. Зайди на [github.com](https://github.com) (создай аккаунт если нет)
2. Нажми `+` → `New repository`
3. Название: `for-her` (или любое другое)
4. Поставь галочку **Public**
5. Нажми `Create repository`

### Шаг 2 — загрузи файлы
**Вариант А — через браузер (проще):**
1. На странице репозитория нажми `Add file` → `Upload files`
2. Перетащи все файлы (`index.html`, `moments.html`, `letter.html`,
   `style.css`, `script.js`) и папку `photos/`
3. Нажми `Commit changes`

**Вариант Б — через Git в VS Code:**
```bash
cd loveweb
git init
git add .
git commit -m "первый коммит"
git remote add origin https://github.com/твой-ник/for-her.git
git push -u origin main
```

### Шаг 3 — включи Pages
1. Зайди в `Settings` репозитория
2. Слева найди раздел **Pages**
3. В Source выбери `Deploy from a branch`
4. Branch: `main`, папка: `/ (root)`
5. Нажми `Save`

Через 1–2 минуты сайт будет доступен по адресу:
```
https://твой-ник.github.io/for-her/
```

## 💡 Советы

- Фото лучше сжать до ~500KB каждое (сайт загрузится быстрее).
  Используй [squoosh.app](https://squoosh.app) — бесплатно, прямо в браузере.
- Для iPhone фото лучше в соотношении 3:4 (портретные).
- Можешь поменять имя в `<title>` каждой страницы — оно показывается
  в браузере и при сохранении на рабочий стол.
